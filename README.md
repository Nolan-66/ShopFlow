## 📌 Description

ShopFlow est une API développée avec FastAPI permettant de gérer :
- Produits
- Panier
- Commandes
- Coupons

Ce projet s’inscrit dans une démarche de **qualité logicielle** avec :
- Tests unitaires et d’intégration
- Couverture de code
- Pipeline CI/CD avec Jenkins
- Analyse de qualité avec SonarQube
